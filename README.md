## Getting Started


### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:wexnox/JS_Framework_CA.git 
   ```
2. Install NPM packages
   ```sh
    npm install
    ```
3. Run the project
    ```sh
    npm start
    ```
